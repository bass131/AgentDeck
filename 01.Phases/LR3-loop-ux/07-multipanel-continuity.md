run·역방향 유령이 사라진다.

## 📚 학습 포인트

- **상태의 수명을 UI 수명에서 분리** — React 컴포넌트가 상태의 *소유자*가 아니라 *구독자*가
  될 때 얻는 것(연속성)과 치르는 것(전역 상태 관리 책임·메모리 상주).
- **진단서 → 테스트 박제** — 재현 절차가 있는 버그는 테스트가 먼저 나온다(TDD의 실전형).
